# Development

## Tech Stack

- [Tauri v2](https://v2.tauri.app/) — desktop shell (Rust)
- [React](https://react.dev/) (latest) — UI framework
- [MUI](https://mui.com/) — component library
- [Monaco Editor](https://github.com/suren-atoyan/monaco-react) — code editor
- [Zustand](https://github.com/pmndrs/zustand) — state management
- [react-i18next](https://react.i18next.com/) — internationalization

## Tooling

- [pnpm](https://pnpm.io/) — package manager
- [Vite](https://vite.dev/) — frontend build tool / dev server
- [TypeScript](https://www.typescriptlang.org/)

## Commands

```bash
pnpm install        # install dependencies
pnpm tauri dev      # run the app in development
pnpm tauri build    # build a production bundle
pnpm dev            # run the Vite dev server only
pnpm build          # type-check and build the frontend
```
